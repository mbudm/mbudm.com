Gallery

A gallery template that let's the content take centre stage.

This is a gallery index module that displays 'thumbnail' links to all images and video items that are in this section. It can also display text blocks, so that you can provide information about your gallery items.


The index module is very customisable or can be left to the default simple grid layout setting. I have customised this example slightly by making some images bigger - so they are 'featured' and I have added this text block.
